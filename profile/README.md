## What is PRIV?

PRIV Protocol is a transparent data economy built on [Base](https://base.org) (Ethereum L2). We connect **consumers** who want to monetize their personal data with **enterprises** who need high-quality, human-generated datasets for AI/ML training, market research, and analytics.

Every transaction is on-chain. Every consent is verifiable. Every contributor gets paid.

## How It Works

| | For Contributors | For Enterprises |
|---|---|---|
| **Browse & Earn** | Earn PRIV passively from desktop and mobile browsing | Access ethically sourced browsing + behavioral data |
| **Data Bounties** | Fulfill specific data requests — photos, video, voice, text | Post bounties for exactly the datasets you need |
| **AI Labeling** | Complete classification, detection, and RLHF tasks | Get human-verified labels for AI model training |
| **Stake & Govern** | Stake PRIV to approve submissions and earn rewards | Community-audited data quality you can trust |

## Tech Stack

- **Blockchain:** Solidity 0.8.24 on Base (Ethereum L2) &middot; 21 deployed & verified contracts
- **Backend:** Next.js 15 &middot; TypeScript (strict) &middot; Supabase (PostgreSQL + Edge Functions)
- **Infrastructure:** Vercel &middot; Upstash Redis &middot; IPFS
- **Payments:** Stripe (SaaS billing) &middot; PRIV Token (on-chain rewards)
- **AI Agent Access:** Model Context Protocol (MCP) server for programmatic data discovery
